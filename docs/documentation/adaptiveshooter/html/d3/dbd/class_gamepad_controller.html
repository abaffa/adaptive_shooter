GamepadController * GamepadController::getNewGamepad ( int  number) [static]

Creates a new controller for the controller number passed.

Parameters:
[in]numberNumber of the controller to get. Range: [0..3]
Returns:
GamepadController* Pointer to controller. NULL if no controller with the number passed. 
{
        int numJoysticks = GameManager::getInstance()->getWindow()->get_ic().get_joystick_count();

        if ( numJoysticks > 0 && number < numJoysticks )
        {
                return new GamepadController( number );
        } 
        else
        {
                return NULL;
        }
